# YouTube Clone (Static Front-End)

### Intro

This project is a static front-end clone of YouTube, including both the Home page and the Video Watch page. The goal is to recreate the layout, structure, and visual behavior of YouTube using only HTML5 and CSS3, focusing on:
- Clean semantic structure
- Component organization
- Accessibility
- Responsive layout
- UI accuracy

---

### Pages Implemented

- Home page (index.html)
- Video watch page (watch.html)

---

### Tech Stack:

HTML5 and CSS3. No frameworks or JavaScript were used.

---

### Features:

Layout & Structure
- Fixed header with search bar
- Sidebar navigation
- Responsive video grid (Home page)
- Structured video layout (Watch page)

Watch Page
- Video section with title, views, and publish date
- Action buttons (Like, Dislike, Share, Save)
- Channel information section
- Description section
- Recommended videos sidebar

UI & Accessibility
- Semantic HTML elements (header, nav, main, aside, section, article)
- Accessible labels using aria-label
- SVG icons for UI elements

---

### Running the Project:

This is a static project, so no build tools are required.

To run it:
1. Clone the repository;
2. Open the folder;
3. Open `index.html` in your browser.
